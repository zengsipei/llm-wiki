# Next.js 技术栈速查

## 框架特性
- App Router：基于文件系统的路由
- Server Components：服务端组件，减少客户端 JS
- API Routes：内置后端 API 支持
- Middleware：请求拦截和修改

## 组件库
- shadcn/ui：基于 Radix UI 的高质量组件（Button、Card、Dialog、Sheet、Form、Table 等）
- Tailwind CSS 4：实用优先 CSS 框架，响应式原生支持

## 数据库
- Prisma：类型安全的 TypeScript ORM
- SQLite：零配置的嵌入式数据库

## 部署命令
- 构建：npx next build
- 生产运行：npx next start
- 开发模式：npx next dev
