    32|## 核心设计理念：Vibe Design（氛围设计）
    33|
    34|- **核心理念**: 从抽象概念和情感出发，通过自然语言描述"用户感受到什么"来生成设计
    35|- **与传统流程对比**:
    36|
    37|| 传统设计工具 | Vibe Design |
    38||------------|-------------|
    39|| 要求画框和选字体 | 描述你希望用户感受到什么 |
    40|| 线框图优先工作流 | 描述业务目标、期望的用户感受或设计灵感 |
    41|
    42|- **使用示例**:
    43|  - *"我希望这个感觉高端而极简，像 Stripe 的网站一样"*
    44|  - *"目标是让用户在 30 秒内完成注册——要有紧迫感但不要让人反感"*
    45|  - *"要有趣味性和色彩感，面向 Z 世代，灵感来源于 Duolingo"*
    46|
    47|- **核心优势**: 探索速度——可以在绘制一个线框图的时间内评估十个设计方向
    48|
    49|## AI 模型驱动
    50|
    51|- **标准模式**: Gemini 2.5 Flash（平衡速度，每月350次生成）
    52|- **实验模式**: Gemini 2.5 Pro（注重质量，每月50次生成）
    53|- **2025年12月更新**: 引入 Gemini 3，提升上下文理解、布局精致度和无障碍访问支持
    54|
    55|## 核心功能
    56|
    57|### 1. Vibe Design（氛围设计）
    58|- 描述用户感受和业务目标，AI 生成多个匹配氛围的设计方向
    59|- 从模糊想法到高保真 UI 的转换
    60|- 不再依赖线框图和组件拖拽
    61|
    62|### 2. Voice Canvas（语音画布）
    63|- 直接对画布说话，AI 助手实时响应
    64|- 倾听需求、提出澄清性问题、给出实时设计评价、进行实时更新
    65|- 适合早期探索阶段，突破文字描述的限制
    66|
    67|### 3. Design Agent（设计智能体）
    68|- 追踪整个项目历史，在不同版本间进行推理
    69|- 理解不同版本间的关系和演进逻辑
    70|
    71|### 4. Agent Manager（智能体管理器）
    72|- 同时探索多个设计方向
    73|- 不会丢失任何分支，支持并行探索不同的设计路径
    74|
    75|### 5. AI 原生无限画布
    76|- 支持图像、文本或代码等多种上下文形式
    77|- 支持从任意 URL 一键提取设计系统
    78|- Direct Edits：直接在画布内手动调整文本、替换图片和微调细节
    79|
    80|### 6. 即时原型（Instant Prototyping）
    81|- 直接在画布上将多个屏幕连接成交互式原型
    82|- 选择两个或更多屏幕，定义流程
    83|- 点击"播放"预览整个用户体验
    84|- 自动生成逻辑后续屏幕（如有登录屏，可自动生成主页屏）
    85|- 从文本提示到可点击多屏原型可在几分钟内实现
    86|
    87|### 7. 图片转 UI
    88|- 上传截图、白板草图照片或任何视觉参考
    89|- 将粗糙草图转化为精致设计稿
    90|- 重新设计现有界面
    91|
    92|### 8. 全局主题管理
    93|- 10+ 预设专业配色方案，或让 AI 自动选择
    94|- 全局主题颜色同步：编辑主题色，所有页面自动更新
    95|- 深度集成 Material Design 3 设计令牌（Design Tokens）
    96|
    97|## 输出与导出
    98|
    99|### 代码导出
   100|支持多种生产级代码格式：
   101|- React/JSX
   102|- HTML/CSS
   103|- Vue
   104|- Flutter
   105|- SwiftUI
   106|- 代码具有功能性和响应式特性，适合直接用于网页项目
   107|- Gemini 3 提升了代码质量，生成更语义化的标记和更有条理的样式表
   108|
   109|### Figma 导出
   110|- 保留正确的 Auto Layout 结构和可编辑图层
   111|- 生成真正的 Figma 文件（非扁平图片）
   112|- 图层有命名、组件按逻辑分组
   113|
   114|### Google AI Studio 集成
   115|- 将 Stitch 设计导入 Google AI Studio
   116|- 添加后端逻辑、API 连接和动态功能
   117|- 从静态 UI 转化为可运行应用
   118|
   119|### Antigravity 集成
   120|- Google 的 AI 驱动 IDE，与 Stitch 深度集成
   121|- 从 GitHub 直接安装 Stitch Skills 到 Antigravity 工作空间
   122|- 使用自然语言提示生成、重新设计和导出前端项目
   123|
   124|### MCP 服务器
   125|- 与 Google 更广泛的生态系统集成
   126|- 支持与其他编码助手兼容：
   127|  - Gemini CLI
   128|  - Claude Code
   129|  - Cursor
   130|
   131|### DESIGN.md
   132|- 通过 DESIGN.md 格式实现设计规则跨平台同步与复用
   133|- 构建自适应语义系统
