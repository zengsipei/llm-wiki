三层命令架构
快捷命令 lark-cli calendar +agenda
快捷命令特点：
+ 前缀，内置智能默认值、表格输出和 dry-run 预览
API 命令 lark-cli calendar calendars list
API 命令特点：
从飞书 OAPI 元数据自动生成，100+ 精选命令
通用 API 调用 lark-cli api GET /open-apis/calendar/v4/calendars
通用 API 特点：
直接调用任意飞书开放平台端点，覆盖 2500+ API
