三层命令架构

第1层：快捷命令（Shortcuts）

+ 前缀，内置智能默认值，适合日常使用

lark-cli calendar +agenda
lark-cli im +messages-send --chat-id "oc_xxx" --text "Hello"
lark-cli docs +create --doc-format markdown --content $'# 标题\n内容'

第2层：API 命令

从飞书 OAPI 元数据自动生成，100+ 精选命令

lark-cli calendar calendars list
lark-cli calendar events instance_view --params '{"calendar_id":"primary","start_time":"1700000000","end_time":"1700086400"}'

第3层：通用 API 调用

直接调用任意飞书开放平台端点，覆盖 2500+ API

lark-cli api GET /open-apis/calendar/v4/calendars
lark-cli api POST /open-apis/im/v1/messages --params '{"receive_id_type":"chat_id"}' --data '{"receive_id":"oc_xxx","msg_type":"text","content":"{\"text\":\"Hello\"}"}'

26个 AI Agent Skills

飞书 CLI 原生为 AI Agent 设计，开箱即用

  • lark-shared
    应用配置、认证登录、身份切换
  • lark-calendar
    日历日程 CRUD、议程查看、忙闲查询
  • lark-im
    发送/回复消息、群聊管理、消息搜索
  • lark-doc
    创建/读取/更新/搜索文档（Markdown）
  • lark-drive
    上传/下载文件，管理权限与评论
  • lark-markdown
    Drive 中原生 Markdown 文件操作
  • lark-sheets
    电子表格 CRUD/追加/查找/导出
  • lark-base
    多维表格、字段、记录、视图、仪表盘
  • lark-task
    任务/任务清单/子任务/提醒/成员分配
  • lark-mail
    邮件收发/草稿管理/监听新邮件
  • lark-contact
    按姓名/邮箱/手机号搜索用户
  • lark-wiki
    知识空间、节点、文档管理
  • lark-event
    实时事件订阅（WebSocket），Agent 友好格式
  • lark-slides
