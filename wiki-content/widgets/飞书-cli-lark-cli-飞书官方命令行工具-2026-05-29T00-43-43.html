飞书 CLI 交互式演示

官方命令行工具 - 18个业务域，200+命令，26个AI Agent Skills

飞书 CLI 是官方开源的命令行工具，让人类和 AI Agent 都能在终端直接操作飞书。覆盖18个业务域、200+命令、26个AI Agent Skills。

安装步骤

1
安装飞书 CLI
npx @larksuite/cli@latest install
2
配置应用凭证
lark-cli config init
3
OAuth 登录授权
lark-cli auth login --recommend

三层命令架构

  • 1️⃣ 快捷命令：+ 前缀，内置智能默认值，适合日常使用
  • 2️⃣ API 命令：从 OAPI 元数据自动生成，100+精选命令
  • 3️⃣ 通用 API 调用：直接调用任意飞书开放平台端点，覆盖2500+ API
