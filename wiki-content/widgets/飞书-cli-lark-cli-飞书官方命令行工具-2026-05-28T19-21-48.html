快速安装
使用 npm 安装飞书 CLI
npx @larksuite/cli@latest install
快速开始
配置应用凭证并登录授权
lark-cli config init
lark-cli auth login --recommend
安全提示：飞书 CLI 支持多种安全特性，包括输入防注入、终端输出净化和原生密钥链存储凭证，确保数据安全。

三层命令架构

⚡ 快捷命令（Shortcuts）
使用 + 前缀，内置智能默认值、表格输出和 dry-run 预览，适合人类和 AI Agent 日常使用
lark-cli calendar +agenda
✓ 预览: 查看今日日程，输出格式化表格
🔧 API 命令
从飞书 OAPI 元数据自动生成，100+ 精选命令与平台端点一一对应
lark-cli calendar calendars list
✓ 预览: 列出所有日历，返回 JSON 格式数据
🌐 通用 API 调用
直接调用任意飞书开放平台端点，覆盖 2500+ API
lark-cli api GET /open-apis/calendar/v4/calendars
✓ 预览: 调用通用 API，返回完整响应数据
