# 示例知识页面

这是一个示例页面，用于展示 LLM Wiki 的页面结构和内容格式。

## 页面结构示例

每个知识页面由 YAML 前置元数据和 Markdown 正文组成。元数据包含页面的唯一 ID、标题、类型、标签和时间戳信息。

## Markdown 格式示例

Wiki 页面支持完整的 Markdown 语法，包括：

- **文本格式**：粗体、斜体、行内代码
- **列表**：无序列表和有序列表
- **代码块**：支持语法高亮的代码片段

```python
# 代码块示例
def hello():
    print("Hello, LLM Wiki!")
```

- **表格**：

| 功能 | 说明 |
|------|------|
| 文档摄入 | AI 自动提取知识点 |
| 智能问答 | 自然语言检索知识 |
| 健康检查 | 定期检测知识问题 |
| 知识图谱 | 可视化页面关联 |
| 导出 | ZIP 压缩包备份 |

## 引用与链接

在知识库中，页面之间通过交叉引用（cross-references）互相链接，形成知识网络。良好的交叉引用是知识库质量的重要指标。
